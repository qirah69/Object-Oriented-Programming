enum class fileType
strong

Represents the specific type of a File component.

This strongly-typed enumeration is used by the File class to distinguish between different kinds of files (e.g., Image files vs. Text files) within the simulated file system.

Enumerator
Image 

Represents an image file (e.g., .png, .jpg).

Text 

Represents a text file (e.g., .txt, .doc).
