\doxysubsection{Detailed Description}
A concrete class representing an individual file in the file system. 

The \doxylink{classFile}{File} class inherits from \doxylink{classFSComponent}{FSComponent} and acts as a "{}leaf"{} node in the composite pattern. Unlike a \doxylink{classFolder}{Folder}, a \doxylink{classFile}{File} cannot contain other components. It stores its own specific size and file type.
