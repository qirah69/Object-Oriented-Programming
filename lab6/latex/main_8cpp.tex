\doxysubsection{Detailed Description}
Entry point for the \doxylink{classFile}{File} System simulation. 

This file contains the execution logic which demonstrates the usage of the composite pattern implemented in the \doxylink{classFile}{File} System architecture. It creates a hierarchical structure of folders and files, displays them, and safely cleans up the allocated memory.
