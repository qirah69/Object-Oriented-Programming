\doxysubsection{Detailed Description}
A composite class representing a directory in the file system. 

The \doxylink{classFolder}{Folder} class inherits from \doxylink{classFSComponent}{FSComponent} and acts as a composite object. It can contain multiple \doxylink{classFSComponent}{FSComponent} objects, allowing it to store both Files and other sub-\/\+Folders. It is also responsible for managing and freeing the memory of the components it owns.
